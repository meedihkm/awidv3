# Changelog

## [1.5.0] - 2026-01-20
### Added
- **Flutter Tests**: Suite complète de tests (Unit, Widget, Integration, Goldens).
- **CI Ready**: Configuration pour execution headless des tests.

## [1.4.0] - 2026-01-19
### Added
- **Security**: Audit OWASP ZAP complet et correction des vulnérabilités.
- **Load Testing**: Scénarios Artillery pour tester la robustesse.
- **E2E Tests**: Tests de bout en bout pour les flux critiques (Commande, Livraison).

## [1.3.0] - 2026-01-15
### Changed
- **API v2**: Refonte complète de l'API vers une architecture modulaire Express/Postgres.
- **Validation**: Ajout de Zod pour la validation rigoureuse des entrées.
- **Logs**: Intégration de Winston et Sentry.

## [1.0.0] - 2025-12-01
### Initial Release
- Version initiale avec fonctionnalités de base (Auth, Commandes simples).
